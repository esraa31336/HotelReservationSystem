# Hotel Reservation System

A Hotel Reservation System built with ASP.NET Core MVC that allows users to browse rooms, make reservations, and manage their bookings, while providing administrators with full 
control over rooms.

## Features

- Authentication & Authorization using ASP.NET Core Identity
- Role-Based Access (Admin & User)
- Room Management (CRUD)
- Room Type Management (CRUD)
- Hotel Reservation
- Reservation Update & Cancel
- Room Availability Validation
- Image Upload
- Dynamic Total Price Calculation
- My Reservations

## Technologies

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- LINQ
- Bootstrap


- `appsettings.json` is not included in this repository for security reasons because it contains the database connection string.
- Create your own `appsettings.json` file and configure your local SQL Server connection before running the project.
